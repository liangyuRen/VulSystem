## 🔧 项目信息

- **项目名**: VulSystem（漏洞管理系统）
- **技术栈**: Java Spring Boot 2.x + MySQL + Flask + Python
- **后端端口**: 8081
- **数据库**: MySQL (数据库名: kulin)
- **主要功能**:
  - 用户认证和管理
  - 项目上传和扫描
  - 漏洞检测和匹配
  - 漏洞报告生成
  - 多语言支持
